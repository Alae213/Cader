# Roadmap

> Generated by setup wizard — March 2026.
> Priorities are opinionated. When scope shifts, update this file and note the reason.

## v1 — Core (MVP)

The minimum set of features for a creator to launch a paid community and a student to join it.

**Foundation**
- [ ] Next.js 14+ App Router scaffold (3 routes: `/`, `/[communitySlug]`, `/help`)
- [ ] Convex schema and indexes (communities, memberships, users, pointEvents, posts, classrooms, modules, pages, lessonProgress, classroomAccess, upvotes, comments, categories)
- [ ] Clerk auth integration (sign-up/sign-in modal, webhook → Convex users sync)
- [ ] Clerk middleware + community membership middleware

**Community**
- [ ] Community creation modal (2-step: name/slug + pricing/Chargily/wilaya)
- [ ] `/[communitySlug]` SPA shell (top bar, tab navigation, conditional visibility)
- [ ] About tab (public, inline editing for owner)
- [ ] Community feed (posts, comments, upvotes, categories, pin, @mentions)
- [ ] Classrooms tab (grid, viewer, module/page tree, slash command editor, progress)
- [ ] Classroom access model (open / level-gated / price-gated / level+price)
- [ ] Members tab + Algeria Wilaya SVG map (interactive filter)
- [ ] Leaderboard tab (5 levels, pointEvents, top 7d/30d/all-time)
- [ ] Analysis tab (owner-only "coming soon" placeholder)

**Payments**
- [x] Chargily Pay integration (community membership payments)
- [x] Chargily Pay integration (classroom price-gated payments)
- [x] Platform subscription billing (2,000 DZD/month via platform's Chargily)
- [x] Webhook handlers (`/api/webhooks/chargily`, `/api/webhooks/clerk`)
- [x] Free tier enforcement (50 members) + locked tier
- [ ] **Security fixes needed**: Webhook signature verification broken, API keys stored unencrypted (T123-T125)

**Modals**
- [ ] Onboarding modal (name + phone + Chargily checkout for paid, webhook-gated)
- [ ] Profile modal (avatar, activity map, communities)
- [ ] Settings modal (profile, notifications, admins, billing, danger zone, account)
- [ ] Create new community modal (reused from landing)
- [ ] Explore modal (community directory)
- [ ] Classroom viewer (within SPA — no route change)

**Infrastructure**
- [ ] Tab persistence (localStorage, access-safe fallback)
- [ ] All 13 edge cases handled
- [ ] `.env.example` and all environment variables documented
